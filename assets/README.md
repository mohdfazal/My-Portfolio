# Profile Image Instructions

## 📸 How to Add Your Profile Picture

### Step 1: Prepare Your Image
- **Format**: JPG, PNG, or WebP (JPG recommended)
- **Size**: At least 400x400 pixels (the larger, the better)
- **Style**: Professional headshot or portrait
- **Background**: Clean, professional background

### Step 2: Add Your Image
1. **Copy your photo** to this `assets` folder
2. **Rename it** to `profile.jpg`
3. **Make sure** it's in the correct location: `assets/profile.jpg`

### Step 3: Test
- Open `index.html` in your web browser
- Your photo should appear in the hero section with beautiful effects

### Troubleshooting
If your image doesn't show up:
1. Check the file name is exactly `profile.jpg`
2. Make sure it's in the `assets` folder
3. Try refreshing the browser (Ctrl+F5)
4. Check the file format is supported
