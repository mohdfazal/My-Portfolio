## 🚀 Getting Started

1. **Clone or download** this repository
2. **Add your profile image**:
   - Place your profile photo in the `assets/` folder
   - Name it `profile.jpg`
   - Recommended size: 400x400 pixels or larger
3. **Open `index.html`** in your web browser
4. **Customize** the content as needed

## 🎨 Customization

### Colors
The website uses an amazing color scheme with vibrant gradients. You can customize colors by modifying the Tailwind config in the HTML file:

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#0f172a',      // Deep blue
                secondary: '#1e293b',    // Medium blue
                accent: '#06b6d4',       // Cyan
                accent2: '#8b5cf6',      // Purple
                accent3: '#f59e0b',      // Orange
                dark: '#020617',         // Very dark blue
                neon: '#00ffff',         // Bright cyan
                purple: '#a855f7',       // Bright purple
                orange: '#f97316'        // Bright orange
            }
        }
    }
}
```
